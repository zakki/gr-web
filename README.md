# gr-web

Web port of Gunroar.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Test

```bash
npm run typecheck
npm run test
```

## Controls

- Arrow keys: move
- Z / X: shot / charge shot
- P: pause

### Mobile controls

- Left side drag: move
- Right side upper tap/hold: shot
- Right side lower tap/hold: charge shot
- Top-right tap: pause
- Touch-capable devices show translucent guide circles for these areas
